## Installation

_Bower_ release installation:

    $ bower install liaison

_Manual_ master installation:

	$ git clone git://github.com/ibm-js/liaison.git

Then install dependencies with bower (or manually from github if you prefer to):

	$ cd liaison
	$ bower install
